#Spell-club

Repozytorium wykorzystywane do projektu na przedmiot Analiza danych z R i Python niest. 2025/26. Członkowie grupy: Igor Korusiewicz, Dominik Łukasiuk, Szymon Chodun
Nasz projekt ma na celu zastosowanie zaawansowanych technik statystycznych i wizualizacyjnych w środowisku Python do zbadania i modelowania zależności między wydajnością graczy NBA a ich zarobkami w sezonie 2024/2025.

$Źródła danych: 
https://www.espn.com/nba/salaries/_/year/2025 -> Zarobki w sezonie 2024/2025
https://www.nba.com/stats/players/traditional?PerMode=Totals&sort=PTS&dir=-1&Season=2024-25 -> Statystyki zawodników w sezonie 2024/2025

#Podział zadań:
1) Czyszczenie i przekształcanie danych. - Igor Korusiewicz, Szymon Chodun 
2) Eksploracyjna analiza danych plus statystyki opisowe. - Szymon Chodun
3) Wizualizacje (wykresy)* - Dominik Łukasiuk
4) Wnioskowanie statystyczne (przedziały ufności, przedziały predykcji, testy, np. Anova, chi-kwadrat albo testy modeli regresji). - Igor Korusiewicz, Dominik Łukasiuk
